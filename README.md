# Gestor de Libros ORM (Taller: ORM + Transacciones + Concurrencia)

## Requisitos
- Python 3.9+
- pip
- (opcional) virtualenv

## Instalación rápida
1. Crear y activar entorno virtual (recomendado)
```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# Unix/macOS
source .venv/bin/activate
```
